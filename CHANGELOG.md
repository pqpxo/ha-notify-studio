# Changelog

## 0.1.20 - Quick-control and logs layout refinement

- Removed the circular border and background around favourite star controls; saved favourites now display as a standalone star icon.
- Rebuilt the Logs screen into a compact controls sidebar and responsive two-column activity view, removing unused page space on desktop.
- Log entries now use the full available content width, with responsive single-column behaviour on smaller screens.
- Updated version metadata, panel cache URL, documentation, and release workflow to `0.1.20`.
