# Notify Studio

Build, test, template, organise, audit, and diagnose rich Home Assistant Companion notifications.

## v0.1.18 highlights

- Added persistent **quick-control favourites** for custom category and area controls.
- The controls area shows one responsive row by default, with an expand/collapse control for all saved entities.
- Use the star in the top-right of a control to pin or unpin it.
- Desktop favourite capacity follows the available viewport width; excess trailing favourites are removed when the layout becomes narrower.
- Mobile keeps up to seven favourite controls in a single horizontal swipe row.
