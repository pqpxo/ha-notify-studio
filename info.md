# Notify Studio

Build, test, template, organise, audit, and diagnose rich Home Assistant Companion notifications.

## v0.1.17 highlights

- Custom category and area buttons are now consolidated into one unified control panel.
- Each bulk or individual entity button continues to show its custom category/area tag.
- **Select entities** now lives in **Manage groups**, beside the matching group's Rename and Delete controls.
- The Notifications page no longer contains a Group by dropdown; sources remain in one filtered grid.
- Manage groups and Scan now share a single desktop action row.
